# ESXi Patching via Lifecycle Manager

## Overview
This project documents the process of patching VMware ESXi hosts using vSphere Lifecycle Manager (vLCM).  
It demonstrates compliance management, baseline creation, and automated patch deployment.

## Steps
1. Create a patch baseline in vSphere Lifecycle Manager.
2. Attach the baseline to ESXi hosts or clusters.
3. Scan hosts for compliance.
4. Stage patches and remediate hosts.
5. Verify patch installation and host stability.

## Tools Used
- VMware vSphere 7.x
- vCenter Lifecycle Manager

## Outcome
- Hosts patched to latest security updates.
- Reduced vulnerability exposure.
- Automated compliance reporting.
